# full-stack-experiment-4
Experiment 4.1 - CLI Employee Management System
Aim
To build a command-line employee management system using Node.js with array-based storage.

Objectives
Implement CRUD operations for employees
Create interactive CLI menus
Validate user input
Handle array operations
Hardware/Software Requirements
Node.js 18+
Terminal/Command Prompt
No additional dependencies
How to Run
node employee-management.js
Features
Add Employee – Enter name, position, and salary
List Employees – View all employee records with ID, name, position, salary
Update Employee – Update employee details by ID
Delete Employee – Remove an employee by ID
Input Validation – Validates name, position, and salary fields
Learning Outcomes
Understanding of CRUD Operations
Learned how to implement Create, Read, Update, and Delete operations using JavaScript and manage data efficiently in an array.
Command-Line Interface (CLI) Development
Gained hands-on experience in building an interactive CLI application using Node.js and handling user inputs through the terminal.
Input Validation Techniques
Developed skills to validate user inputs (name, position, salary) to ensure data accuracy and prevent invalid entries.
Data Handling Using Arrays
Improved understanding of array manipulation methods such as adding, updating, searching, and deleting records.
Program Flow Control & Logic Building
Strengthened problem-solving skills by designing menu-driven logic, conditional statements, and loops for smooth program execution.
